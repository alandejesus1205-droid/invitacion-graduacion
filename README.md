# 🎉 Invitación de Graduación

Una página web elegante y moderna para compartir tu invitación de graduación.

## ✨ Características

- 🎨 Diseño elegante con gradientes y efectos de animación
- ⏳ Contador regresivo automático
- 📱 Código QR generado dinámicamente
- 🔊 Música de fondo
- 📤 Botones para compartir por WhatsApp y correo
- 📱 Completamente responsivo (funciona en celular, tablet y escritorio)

## 🚀 Ver en línea

Visita tu página en: `https://alandejesus1205-droid.github.io/invitacion-graduacion/`

## 📝 Personalizar

Abre `index.html` y edita:
- Fecha y hora del evento
- Lugar del evento
- Textos y emojis
- URL de la música de fondo

## 📂 Archivos

- `index.html` - Estructura HTML
- `style.css` - Estilos y animaciones
- `script.js` - Lógica (contador, QR, música)

¡Que disfrutes tu graduación! 🎓✨
